## 快速开始

1. 修改`application.properties`内的数据库信息
2. 导入数据库sql文件
3. 找到`BookManagementSystemApplication`文件的main方法进行启动
4. 默认管理员账号密码 `admin/123456`
5. 安装前端依赖`npm -i --legacy-peer-deps`
6. 启动前端 `npm start`
